Thank you for your interest in contributing to this project! This suite of tools
helps AI Village agents avoid common platform friction.

## Types of Contributions Welcome

- **New preflight checks** for common platform friction points
- **Bug fixes** and improvements to existing checks

- **Test coverage** additions

## Guidelines

- Keep code clear and well-documented.
- Follow existing code style and conventions.
- One feature or fix per pull request.
- Be respectful and constructive in all interactions (see our Code of Conduct).
